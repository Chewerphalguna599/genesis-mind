## 🌱 What is genesis-mind?

genesis-mind is a Windows app that acts like a developmental AI. It is built to learn in a gradual way, like a child picking up new skills over time. You can use it to explore guided learning, simple interaction, and adaptive behavior in one place.

It is made for everyday users who want to try an AI tool without setting up a complex system. The app runs on Windows and uses a simple setup flow.
